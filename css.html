Welcome! CSS.

CSS stands for Cascading Style Sheets.
It can control the layout of multiple web pages all at once.
CSS = Styles

CSS can be added to HTML documents in 3 ways: Inline, External

Inline - inside HTML elements

Example:

h1 style="color:blue;

Internal -in the section

Example:

<head>
<style>
body {background-color: powderblue;}
h1 {color: blue;}
p {color: red;}
</style>
</head>

link

External - by using a <link> element to link to an external CSS file

Example:

<head>
<link rel="stylesheet" href="styles.css">
</head>


CSS Color, Font-family, font-size

CSS color = the text color
CSS font-family = font
CSS font-size =the text size

Example:

p {
color: red;
font-family: courier;
font-size: 160%;
}

Example CSS Border

p {
border: 2px solid powderblue;
}

Example CSS Padding

space between the text and the border.

p {
border: 2px solid powderblue;
padding: 30px;
}

Example CSS Margin

space outside the borde.

p {
border: 2px solid powderblue;
margin: 50px;
}
