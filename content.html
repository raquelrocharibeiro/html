01-HTML Home

HTML is the standard markup language for creating Web pages.

02-HTML Introduction

HTML stands for Hyper Text Markup Language, for creating Web pages. HTML describes the structure of a Web page and consists of a series of elements HTML elements tell the browser how to display the content and label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

03-HTML Editors

Web pages can be created and modified by using professional HTML editors. However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac), is a good way to learn HTML.

04-HTML Basics

All HTML Documents must start with a document type declaration:<!DOCTYPE html> and helps browsers to display web pages correctl. The HTML document itself begins with <html> and ends with <html>, and the visible its part is between <body> and </body>.
