accesskey

Global Attribute
Specifies a shortcut key to activate/focus an element

<a href="https://www.w3schools.com/html5" accesskey="h">HTML5</a><br>
<a href="https://www.w3schools.com/css3" accesskey="c">CSS3</a>

async

Specifies that the script is executed asynchronously (only for external scripts)
When present, it specifies that the script will be executed asynchronously as soon as it is available.

<script src="demo_async.js" async></script>


charset

Specifies the character encoding
use in meta or script

<head>
<meta charset="UTF-8">
</head>

<script src="myscripts.js" charset="UTF-8"></script>


content

Use in meta.
Specifies the value associated with the http-equiv or name attribute

<head>
<meta charset="utf-8">
<meta name="description" content="free web tutorials">
<meta name="keywords" content="html, css, javascript">
<meta name="author" content="john doe">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>


http-equiv

Values: content-security-policy, content-type, default-style, refresh
The http-equiv attribute provides an HTTP header for the information/value of the content attribute.
can be used to simulate an HTTP response header.

<head>
<meta http-equiv="refresh" content="30">
</head>

contenteditable

attribute specifies whether the content of an element is editable or not.

<element contenteditable="true|false">


data

Specifies the URL of the resource to be used by the object
use in object element.

<object width="400" height="400" data="helloworld.swf"></object>

example below
<object data="images/laura.jpeg" width="300" height="200">


data-*

The data-* attribute consist of two parts:
The attribute name should not contain any uppercase letters, and must be at least one character long after the prefix "data-"
The attribute value can be any string
Custom attributes prefixed with "data-" will be completely ignored by the user agent


<!DOCTYPE html>
<html>
<head>
<script>
function showDetails(animal) {
var animalType = animal.getAttribute("data-animal-type");
alert("The " + animal.innerHTML + " is a " + animalType + ".");
}
</script>
</head>
<body>

<h1>Species</h1>
<p>Click on a species to see what type it is:</p>

<ul>
<li onclick="showDetails(this)" id="owl" data-animal-type="bird">Owl</li>
<li onclick="showDetails(this)" id="salmon" data-animal-type="fish">Salmon</li>
<li onclick="showDetails(this)" id="tarantula" data-animal-type="spider">Tarantula</li>
</ul>

</body>
</html>


datatime

specifies the date and time when the text was deleted/inserted.
The time element does not render as anything special in any of the major browsers.

The datetime attribute can be used on the following elements:del, ins, time

<ins datetime="2015-09-15T22:55:03Z">This is an inserted text.</ins>
